# MiniGame-Python
A simple Space Shooting for 2 players (same PC).

# Usage
run **main.py** to play game

# Keys
## Player 1: 
- Move: W, A, S, D
- Shoot: F
## Player 2:
- Move: Arrow Up, Arrow Down, Arrow Left, Arrow Right
- Shoot: Numpad-0 (if you don't have numpad keys then sorry :( )
